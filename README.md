# Shadowfox
Personal Portfolio webpage using HTML and CSS as a part of Shadowfox internship (Task 1)
It showcases my skills, projects, certifications, education, and contact details in a clean, responsive, and visually appealing layout.
The portfolio is designed to represent me as a Web Development Intern / Student, highlighting both technical knowledge and creativity.

# Features
Responsive design (works on desktop, tablet, and mobile)
Modern and attractive UI
Smooth hover effects and transitions
Well-structured sections
Clean and readable code

# Sections Included
Home – Brief introduction and profile
About Me – Overview of who I am and my interests
Skills – Technical skills related to web development
Projects – Showcasing my academic and personal projects
Certifications & Courses – Online courses and achievements
Education – Academic background
Contact – Email, phone number, and social links
